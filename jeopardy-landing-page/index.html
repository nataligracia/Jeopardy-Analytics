Wondering how you can use Jeopardy questions and answers to advance Natural Language Processing research?

Why Jeopardy?

Jeopardy provides a vast amount of data that not only include data on how questions have changed over the years, but provided an opportunity to research and understand how natural language processing can be implemented to better classify bodies of text, no matter how limited in scope.

Findings

Identifying and utilizing the most informative segment of available data (questions versus answers) to classify topics is crucial prior to using any machine learning or natural language processing tool.

Future Uses

The findings from this project can not only be used by a Jeopardy enthusiast to see which topics are most prevalent and consistently asked, but can also be used as a starting point to better classify customer reviews, standardized test questions and more!

Earnings Per Show Increase Significantly in the Early 2000s

There has been a significant increase in the earnings per show since the early 2000s. Some of this can be attributed to all-time highest Jeopardy earner Ken Jennings in 2005!

Certain Categories Are More Popular Than Others

It appears as though overtime, the same type of questions have been the most frequently asked in Jeopardy, but these categories do not provide much of a view into the actual questions within each categories.

Questions Can Be Better Categorized Using Machine Learning

Even with a limited corpus that each Jeopardy question and answer provide, we can use an unsupervised Natural Language Processing model to better assign general category names to each question. For example, of a our model was able to classify over 500 samples into a general music category.
